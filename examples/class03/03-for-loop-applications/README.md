# For-Loop | Applications

## by count
We just need to tell the computer **how many times to repeat!** A loop isn't useful without some **variation**, so adding it is very important. Since `i` changes every time, you can use it to change **sizes**, **colors**, and **more**.

## by range
You can also give the computer a **range**, like telling it to go **from here to there**. Since we're covering a range (distance) instead of just counting, taking a **bigger step** like `x += 20` instead of `i++` is really key here! And remember, you can use **that changing value** `x` to create some fun visual changes.
